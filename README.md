# 📰 Fake News Detection System

A Machine Learning-based web application that classifies news articles as **REAL** or **FAKE** using Natural Language Processing and Logistic Regression.

---

## 🚀 Project Overview

With the rapid spread of misinformation online, detecting fake news has become crucial.  
This project builds a predictive model that analyzes news content and determines its authenticity.

---

## 🧠 Tech Stack

- Python
- Scikit-learn
- Pandas, NumPy
- NLP (Text Preprocessing)
- TF-IDF Vectorization
- Logistic Regression
- Streamlit (for web app)

---

## 📊 Dataset

The dataset used in this project is taken from **Kaggle**, containing labeled news articles categorized as real or fake.

---

## ⚙️ Features

- Clean and preprocess text data
- Convert text into numerical features using TF-IDF
- Train Logistic Regression model
- Real-time prediction using Streamlit interface
- User-friendly UI for testing news content
